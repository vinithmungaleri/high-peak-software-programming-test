# high-peak-software-programming-test

Download as zip file and extract to your computer.
code_recording.mp4 contains the video of code execution.
In input files folder each test case is given as seperate input file such as input1.txt for 1st test case ..etc
Open the folder and click on python_code_file.ipynb file (This is a jupyter notebook file format).
Run the code by either pressing control+enter or click on run in menu.
you can see the output files generated for each input file in output files folder.
